# 🌱 Project Almond — Interactive Lawn

브라우저 전체를 잔디밭으로 채우는 인터랙티브 웹 실험 프로젝트입니다.  
마우스의 움직임에 따라 잔디가 흔들리고,  
때때로 나비가 날아오르는 작은 세계를 만듭니다.


---

## ✨ Features

- 풀스크린 잔디밭 렌더링 (Canvas)
- 마우스 이동에 따른 잔디 흔들림
- 드래그로 잔디 뽑기 & 시간에 따른 재성장
- 움직임에 반응해 등장하는 다양한 나비
  - 색상, 크기, 비행 패턴이 서로 다름

---

## 🧠 Concept

이 프로젝트는 정적인 UI가 아니라  
**살아 있는 배경**을 만드는 실험입니다.

- 버튼이 없어도 인터랙션이 존재하고
- 명확한 목적 없이도 만지작거릴 수 있으며
- 가만히 보고 있어도 작은 변화가 일어납니다

웹을 하나의 **작은 생태계**처럼 다루는 것을 목표로 합니다.

---

## 🛠 Tech Stack

- Vanilla JavaScript
- HTML5 Canvas
- Firebase Hosting

외부 라이브러리 없이 구현하는 것을 기본 원칙으로 합니다.
